# PalaeontologyVR

## Overview
This project is a Virtual Reality experience developed in Unreal Engine 5.5 that explores science communication on the topic of palaeontology through immersive, interactive environments. It is being created as part of my Master’s Thesis, aiming to combine scientific accuracy with immersive VR to enhance public interest and understanding in palaeontology.

Design Document: https://app.gitbook.com/invite/IFgcZsUOWEPaxdPHtVvf/tIHscQihzCkrjuHinwcO

Research Blog: https://www.immersivepalaeoart.com/blog

## Objectives
- Present accurate paleontological reconstructions in an interactive 3D environment.
- Enable users to explore extinct organisms and their environments intuitively through VR.
- Evaluate how immersive visualisation can improve the experience of visualising extinct lifeforms and environments.
- Explore the capabilities of VR in displaying high fidelity rigged meshes and realistic environments.

## Project Setup for Development with Meta Quest VR Headset and Windows
1. Install Unreal Engine 5.5
2. Install MetaXR version 78.0, download and follow the instructions on this webpage. https://developers.meta.com/horizon/downloads/package/unreal-engine-5-integration
3. Install Meta Horizon on your phone, enable developer mode for your headset in this app.
4. Install Meta Quest Link on your computer. https://www.meta.com/en-gb/help/quest/1517439565442928/?srsltid=AfmBOoqr7W-BAmwaQoTj3TuE0XFoO5eiimTXuNq2GQDj4dBR7mvI_-Pj
5. Install Meta Quest Developer Hub on your computer. https://developers.meta.com/horizon/downloads/package/oculus-developer-hub-win/
6. Enable Meta XR plugin inside Unreal Engine, then restart Unreal Engine.
7. Connect your Meta Quest headset to your PC via usb cable and link it to your PC inside the headset via the link menu.
8. If Unreal Engine is crashing with the Meta XR plugin and you have Steam VR installed, go to settings in SteamVR, OPenXR settings, Make sure both facial tracking and MR are turned off.
